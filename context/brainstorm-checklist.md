# Brainstorm Checklist

Living list of topics that need an individual, focused brainstorm (each dispatched to a subagent).
When a brainstorm surfaces a new topic worth its own session, add it here and work incrementally.

Output convention: each brainstorm writes to `context/brainstorms/NN-<slug>.md` with sections:
Problem framing / Options with trade-offs / Recommendation / Open questions for the developer /
New brainstorm topics surfaced.
